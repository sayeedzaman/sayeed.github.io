About me

Hi, I'm Md. Sayeeduzzaman, a Computer Science and Engineering graduate from Bangladesh University of Engineering and Technology (BUET). I currently work as an Associate Software Engineer at Bengal Mobile QA Solutions, specializing in Android application development with Java and Kotlin.

My professional experience includes developing the BRAC UPG mobile application with comprehensive API integration and ObjectBox database implementation, building behavior-based malware detection systems, and implementing activity monitoring features. I've also worked on full-stack web development, creating the Khalis Beauty e-commerce platform using modern technologies like NextJS, NodeJS, and PostgreSQL.

My undergraduate thesis focused on Detecting Objects using Knowledge Distillation, where I trained and compared multiple models including YOLO v10, YOLO v8m, RetinaNet, and FasterNet across various datasets. I'm passionate about distributed systems, human-computer interaction, real-time data processing, and machine learning.

When I'm not coding or working on research, I enjoy exploring new technologies and building meaningful projects that solve real-world problems.
